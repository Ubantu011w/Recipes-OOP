## How does it work
A menu with 3 options and two sub-menu.
* Manage ingredients
   * Create an ingredient
   * List all ingredients
   * Get info
   * Delete an ingredient
* Manage recipes
   * Create a recipe
   * List all recipies
   * Get info
   * Delete a recipe
   * Search with price or ingredients
      * Search with price
      * Search with ingredient
* Save and quit
